# 京喜盒子 Swiss IKB — Design Specification

## I. Template Overview

Fidelity replica of the `ppt/index.html` Swiss deck: Carbon spacing rhythm, Klein International Blue (`#002FA7`) accent, alternating light (`#FAFAF8`) and dark (`#0A0A0A`) surfaces, minimal chrome header (section label + page fraction). Suited for 京喜盒子 / 小乐机器人 solution pitches and product storytelling.

## II. Color Scheme

| Role | HEX |
|------|-----|
| Paper (light bg) | `#FAFAF8` |
| Ink (dark bg / text) | `#0A0A0A` |
| Accent IKB | `#002FA7` |
| Grey panel | `#F0F0EE` |
| Border subtle | `#E0E0E0` |
| Text secondary | `#525252` |
| Text helper | `#737373` |
| On accent | `#FFFFFF` |

## III. Typography

Default stack: `Microsoft YaHei, PingFang SC, Arial, sans-serif`. Display titles use weight 200–300, large negative letter-spacing. Meta labels uppercase tracking.
